Packer files for Windows 2016
=============================================

A packer files for a minimal Windows 2016 virtual machine

Packages Included
-----------------

 * Chocolatey
 * OpenSSH
 * Chrome
 * 7zip
 * GIT
 * Visual Studio Code
 * Visual Studio Code PowerShell Extension
